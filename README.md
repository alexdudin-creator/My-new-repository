# RBC460 Tracker

Этот проект автоматически:
- Получает цену фонда RBC460
- Сохраняет её в Excel
- Отправляет уведомление на email
- Работает через GitHub Actions 24/7

## Запуск
Проект запускается автоматически каждый день в 23:00 по Торонто.

## Файлы
- tracker.py — основной скрипт
- tracker.yml — расписание запуска
